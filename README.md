# Todo-React

## Setup

```
$ npm install
```

## Serve for a dev server. Navigate to http://localhost:4200/.

```
$ npm run start
```

## Build the project

```
$ npm run build
```

## Docker Build

```
$ docker build .
$ docker run -p 3000:3000 -e PORT=3000
```

## Cloud Run

```
$ gcloud run deploy
```
